# Controlador Home

El controlador `Home` es responsable de manejar la página principal del sistema.

## Configuración

Este controlador extiende de `BaseController` y utiliza las configuraciones básicas proporcionadas por el framework.

## Funciones

### index()

Renderiza la vista de bienvenida (`welcome_message`).

## Constructor

El constructor de este controlador no tiene configuraciones adicionales específicas.
